# 2026-07-21 AI Daily Briefing

## 핵심 요약

- 최신 arXiv 공개분에서는 **agent skill 생태계의 대규모 정제·검색, MCP 기반 장기 tool-use RL, 실행 가능한 world model과 replay 검증의 효과 분리**가 핵심 주제로 나타났다.
- SkillCorpus는 약 82.1만 개 공개 skill을 9만6,401개로 정제했지만, 성능 향상에는 corpus coverage뿐 아니라 skill을 실제 행동으로 연결하는 harness가 경계 조건으로 작용한다고 보고했다.
- long-context serving에서는 token을 삭제하지 않고 salience에 따라 low-rank 예산을 배분하는 VarRate가 20% KV 예산에서 비압축 모델 대비 0.8점 이내를 유지했다고 저자들이 보고했다.
- NVIDIA는 SIGGRAPH에서 MCP 기반 creative-agent 연결, synthetic-video detector, 4B Cosmos 3 Edge를 한꺼번에 공개하며 agent·physical AI를 local/edge 배포 스택으로 밀고 있다.
- 산업 측면에서는 장시간 agent의 안전·정렬, stateless MCP 운영, 전력당 추론 효율, 제약 산업의 전용 AI factory가 주요 경쟁축으로 부상했다.

## 읽을거리/추적할 점

- SkillCorpus가 공개되면 중복 제거 전후 성능, malicious skill·prompt injection filter의 recall, 다른 harness로 옮겼을 때 +7.5%p 이득이 유지되는지 확인할 것.
- ToolVerse는 MCP server의 실제 version·schema 변화와 destructive tool을 어떻게 격리했는지, GUST가 unseen MCP로 일반화하는지 살펴볼 것.
- VarRate는 vLLM·SGLang 같은 serving engine에서 batch size, context length, GPU별 end-to-end latency·throughput·quality Pareto를 재현할 것.
- ARC-AGI-3 결과는 held-out game과 동일 total action·token·wall-clock budget에서 text baseline과 verification treatment를 다시 비교할 것.
- Cosmos 3 Edge는 VANTAGE-Bench 외에 real robot의 closed-loop success, intervention rate, thermal throttling 하 15Hz 지속 여부를 추적할 것.
- stateless MCP가 session 복구를 단순화하는 대신 replay·duplicate side effect를 키우지 않는지 idempotency key와 audit log 설계를 확인할 것.
- Frozen v2는 익명 소식통 기반 2028년 계획이므로 Google의 공식 제품 발표와 독립적인 tokens/Joule 수치가 나오기 전까지 확정 사실로 취급하지 말 것.
