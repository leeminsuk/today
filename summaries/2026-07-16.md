# 2026-07-16 AI Daily Briefing

## 핵심 요약

- 최신 논문은 coding agent의 도구 반환값 통합을 위한 중간학습, 문서 전용 시각 사전학습, 모든 시각 작업을 RGB 입출력으로 통합하는 방식처럼 **범용 모델의 인터페이스와 학습 구조 자체**를 다시 설계하고 있다.
- Function-aware FIM은 26억 토큰 규모 중간학습만으로 SWE-Bench Verified를 모델별 2.8~3.2점 높여, agent 성능이 post-training뿐 아니라 foundation-model 단계의 학습 목표에도 크게 좌우됨을 보였다.
- OpenAI의 GPT-Red는 self-play 기반 자동 red teaming을, IBM은 agent model routing을 비용·품질·지연·거버넌스가 얽힌 시스템 최적화 문제로 제시해 **운영 harness와 안전성**이 제품 경쟁의 중심으로 이동하고 있음을 보여준다.
- Thinking Machines의 Inkling은 약 1조 파라미터·100만 토큰 context의 open multimodal MoE를, Mistral과 NVIDIA는 각각 단일 카메라 navigation 모델과 Blackwell 기반 edge module을 공개해 embodied AI stack 경쟁을 확대했다.

## 읽을거리/추적할 점

- Function-aware FIM의 968개 repository 비오염 판정과 SWE-Bench contamination audit를 확인하고, Python 외 언어에서도 개선이 유지되는지 재현할 것.
- MonkeyOCRv2의 1억1,300만 이미지 corpus 구성·license와 photographed document에서의 언어별 성능 편차를 확인할 것.
- RINO가 discrete label을 RGB로 바꾸면서 생기는 색상 ambiguity와 정밀도 손실, unseen task 조합의 generalization을 추적할 것.
- SpectraReward가 prompt를 잘 읽어내지만 시각적 품질은 낮은 reward hacking을 유발하지 않는지 human preference와 함께 비교할 것.
- GPT-Red의 threat model·평가 수치·재현 가능한 benchmark가 추가 공개되는지, self-play 공격이 실제 prompt injection 분포로 전이되는지 볼 것.
- Inkling은 open weight 여부와 별개로 license, 최소 hardware, NVFP4 대비 BF16 품질 및 100만 context의 실제 memory·latency를 검증할 것.
- Robostral Navigate 76.6%는 Mistral 공개 수치이므로 R2R-CE 설정과 독립 real-world test를 확인하고, Jetson T3000·T2000은 가격·전력·availability를 추적할 것.
- IBM routing 사례는 특정 417-task agent workload 결과이므로 model version·cache policy가 바뀔 때 총비용 순위가 유지되는지 재측정할 것.
