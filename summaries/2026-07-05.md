## 읽을거리/추적할 점

- `Distributed Attacks in Persistent-State AI Control`은 코딩 에이전트 운영에서 “PR별 리뷰”가 아니라 **세션·시간·상태를 넘는 공격 모델**을 어떻게 감시할지 검토할 가치가 있다.
- `ReContext`와 `Online Safety Monitoring`은 둘 다 모델 자체 변경보다 **하네스/모니터링 레이어**를 강화하는 접근이므로, 운영 시스템 설계 관점에서 비교해볼 것.
- Hugging Face/Cerebras 음성 AI와 DeepMind-A24 협력은 멀티모달 생성 모델의 병목이 모델 품질뿐 아니라 **지연시간, 저작권, 워크플로 통합**으로 이동하고 있음을 보여준다.
- Alibaba의 Claude Code 금지 보도는 기업별 AI coding agent 허용 정책, 코드 반출 통제, 감사 로그 요구사항을 추적해야 할 신호다.
