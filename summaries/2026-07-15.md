# 2026-07-15 AI Daily Briefing

## 핵심 요약

- 7월 13일 공개 논문들은 멀티에이전트의 탐색 실패, 비디오 답변의 픽셀 단위 근거, 재사용 가능한 post-training 신호처럼 **성능 수치보다 검증 가능성과 학습 효율**을 직접 다뤘다.
- AdvancedMathBench에서 최고 모델도 대학원 자격시험 증명 66.1점, 증명 오류 검증 Balanced F1 65.1에 머물러 고급 수학에서는 생성뿐 아니라 **틀린 증명을 가려내는 능력**도 병목으로 확인됐다.
- NVIDIA는 agent skill·지속 메모리·실험 원장을 결합한 RL autoresearch 사례에서 Qwen3-VL-2B-Instruct의 시각 계수 정확도를 25.0%에서 96.9%로 높였다고 공개했다.
- Google은 agent를 격리 VM에 넣고 원시 credential 접근을 막는 CAPSEM을 오픈소스로 발표해, prompt injection을 모델 계층이 아닌 runtime 격리로 제한하는 흐름을 강화했다.
- Anthropic과 Google의 최신 제품은 교사 업무와 브라우저로 agent 기능을 확장하면서, 교육 데이터 보호·민감 작업 확인·prompt injection 방어를 제품 요건으로 전면화했다.

## 읽을거리/추적할 점

- MACE가 agent 수·능력 차이가 커지고 통신 비용이 제한된 실제 coding/research team에서도 coordination regret를 줄이는지 공개 코드로 확인할 것.
- E-VQA의 pixel evidence가 답을 사후 합리화한 mask가 아니라 실제 추론에 사용된 근거인지 causal intervention 평가를 추적할 것.
- PUST의 weak-to-strong transfer가 Qwen3 계열 밖의 architecture와 안전 정렬에서도 유지되는지, proxy 탐색 비용을 포함한 총비용을 비교할 것.
- AdvancedMathBench 자동 verifier의 human agreement 수치와 false-negative 분포를 확인하고, benchmark verifier 자체가 모델 순위를 왜곡하지 않는지 볼 것.
- NVIDIA autoresearch의 25.0%→96.9% 결과는 자체 생성 visual counting 환경의 task-specific 결과이므로 held-out 분포·seed·총 GPU 시간 재현이 필요하다.
- Claude for Teachers pilot에서는 교사 시간 절감뿐 아니라 학생 성취도, 오류 전파, privacy incident를 함께 측정하는지 추적할 것.
- CAPSEM의 소스·위협 모델·VM startup 비용과 A2A/AP2 적용 예제가 공개되는지 확인할 것.
