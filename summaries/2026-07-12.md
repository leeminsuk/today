# 2026-07-12 AI Daily Briefing

## 핵심 요약

- 주말 arXiv 최신분(7월 9일 제출)에서는 **데이터 과학 agent의 인과 추론, 선형 attention, 비디오를 통한 추론, 모바일 depth 추정**이 주요 흐름으로 나타났다.
- CausalDS와 OpenCoF는 각각 도구를 쓰는 데이터 과학 agent와 생성 비디오의 추론 능력을 분리해 평가하려는 시도로, 결과 점수만이 아니라 중간 과정 검증의 중요성을 보여준다.
- 업계에서는 Mistral의 단일 카메라 로봇 내비게이션, OpenAI의 GPT-5.6 및 코딩 benchmark 신뢰성 분석, Hugging Face·vLLM의 모델 통합 단순화가 주목된다.
- 인프라 쪽에서는 host offloading과 런타임 graph 최적화가 HBM 용량 및 모델별 커스텀 구현 병목을 줄이는 실용적 해법으로 제시됐다.

## 읽을거리/추적할 점

- CausalDS가 인과 추론 오류와 도구 사용 오류를 어떻게 분리 채점하는지, 공개 harness의 재현성을 확인할 것.
- Robostral Navigate의 76.6%가 실제 로봇·동적 장애물·조명 변화에서도 유지되는지와 모델·데이터 공개 범위를 추적할 것.
- GPT-5.6은 독립 benchmark와 실제 agent workload에서 비용 대비 성능, latency, 안정성을 비교할 것.
- vLLM의 자동 fusion이 지원하지 않는 architecture와 커스텀 port 대비 성능 편차를 확인할 것.
- NVIDIA의 최대 57% 수치는 GB200 NVL72 및 특정 대형 모델 조건이므로 다른 interconnect·batch·sequence 설정에서의 손익분기점을 검증할 것.
