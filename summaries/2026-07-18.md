# 2026-07-18 AI Daily Briefing

## 핵심 요약

- 최신 연구는 agent의 성능을 단순 정답률이 아니라 **검색 궤적의 인과적 기여, 비용, 장기 context, 오염 가능성**까지 포함해 평가하려는 방향으로 이동하고 있다.
- Bridge Evidence는 agent가 다음 검색어를 찾도록 돕는 문서가 정적 RAG 평가에서는 무용해 보일 수 있음을 실험으로 보였다. agentic retrieval에는 문서 단위 relevance가 아닌 trajectory 단위 평가가 필요하다.
- RoboTTT는 robot policy의 context를 8K timestep으로 늘리면서 inference latency 증가 없이 장기 작업 성능을 높였다고 보고해, context length를 embodied model의 새로운 scaling 축으로 제시했다.
- 업계에서는 Meta가 Anthropic에 최대 100억 달러 규모의 compute를 임대하는 방안을 논의 중이라는 보도가 나왔다. 확정 계약은 아니지만 hyperscaler 밖 AI 기업 간 infrastructure 거래가 커지는 신호다.
- NVIDIA·Hugging Face는 post-training과 대규모 diffusion fine-tuning을 반복 가능한 infrastructure로 제품화하고 있으며, OpenAI는 AI ROI를 성공 task당 비용·신뢰성·compute 수익으로 측정하자고 제안했다.

## 읽을거리/추적할 점

- Bridge Evidence의 문서 삭제 후 재실행이 model stochasticity를 충분히 통제했는지, 한국어 web search에서도 정적 relevance와 CTU의 불일치가 재현되는지 확인할 것.
- SearchOS의 WideSearch·GISA 비교에서 동일 model, 검색 API, token/tool budget, retry policy가 고정됐는지와 Evidence Graph citation 정확도를 볼 것.
- 보안 agent는 공격·방어를 같은 aggregate score로 합치지 말고 inference spend, tool spend, SOC telemetry 접근 비용을 분리해 측정할 것.
- RoboTTT의 87%·62%는 저자 보고값이다. 8K fast-weight update의 memory·stability, unseen robot과 safety-critical perturbation에서의 성능을 독립 검증할 것.
- HalfLife의 inclusion 추정이 실제 상용 pretraining pipeline과 얼마나 일치하는지, discussion content provenance를 crawl 단계에서 보존·차단할 방법을 추적할 것.
- Meta–Anthropic 거래는 논의 단계다. 최종 계약 여부, capacity·기간·data boundary와 Anthropic의 기존 AWS·Google Cloud 관계 변화를 확인할 것.
- NVIDIA의 30% CPU throughput과 2초 미만 weight sync는 공급자·partner 공개 수치다. workload 정의, network topology, energy·총비용을 포함해 비교할 것.
- 정치적 표현 평가는 prompt 언어·국가별 결과와 model version, refusal 기준, 원 보고서의 재현 자료 공개 여부를 확인할 것.
