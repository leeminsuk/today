# 2026-07-13 AI Daily Briefing

## 핵심 요약

- 주말에는 arXiv 신규 배치가 없어, Hugging Face Daily Papers에서 확인된 **7월 9일 최신 제출분 중 기존 브리핑과 겹치지 않는 논문**을 골랐다.
- 연구 흐름은 다중 객체 비디오 분할의 실시간화, 장시간 event-video 복원, 기하 인지형 파노라마 생성, 질환 조건부 분자 생성처럼 **멀티모달 모델의 실제 제약 해결**에 집중됐다.
- 업계에서는 attention kernel을 profiler로 검증하는 실무, Claude의 공공 의견 수렴, OpenAI의 생물안전 red teaming, LeRobot의 개방형 로봇 stack 확대가 눈에 띈다.
- 모델 자체 성능뿐 아니라 실행 trace, 안전성 검증, 공개 데이터·도구, 실제 하드웨어에서의 처리량이 제품 경쟁력의 핵심 축으로 굳어지고 있다.

## 읽을거리/추적할 점

- SAM-MT의 `>36 FPS`가 사용 GPU, 해상도, target 수, occlusion 조건이 바뀌어도 유지되는지 공개 코드로 재현할 것.
- LongE2V와 파노라마 생성 연구는 장시간·360도 일관성을 정량화한 평가셋과 실패 사례를 함께 확인할 것.
- DrugGen 2는 생성 분자의 유효성·합성 가능성·질환 관련성이 wet-lab 검증까지 이어지는지 구분해서 볼 것.
- PyTorch SDPA는 backend 자동 선택 결과를 profiler에서 확인하고, latency뿐 아니라 peak memory와 numerical difference도 비교할 것.
- Anthropic의 공개 질문 initiative와 OpenAI Bio Bug Bounty가 실제 조치·수정·보상·후속 보고서로 이어지는지 추적할 것.
