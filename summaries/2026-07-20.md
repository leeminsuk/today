# 2026-07-20 AI Daily Briefing

## 핵심 요약

- 주말 arXiv 최신 공개분(7월 17일)에서는 **검색 agent의 상태 외부화, hindsight skill 기반 agent RL, world-action model의 행동·상상 불일치 공격, harness 자동 진화 평가의 과적합**이 핵심 주제로 부상했다.
- SearchOS는 frontier task·evidence graph·coverage map·failure memory를 공유 상태로 두고, 검색 실패의 반복과 citation 누락을 system-level harness로 줄이는 방향을 제시한다.
- Hugging Face는 악성 dataset이 processing worker의 두 code-execution 경로를 악용한 뒤 autonomous agent swarm으로 내부 cluster에 침투한 사건을 공개했다. 공개 model·dataset·Space 변조 증거는 없지만 사용자에게 token rotation과 활동 점검을 권고했다.
- NVIDIA·Hugging Face는 Diffusers checkpoint를 변환하지 않고 NeMo Automodel에서 image/video diffusion model을 single GPU부터 multi-node까지 fine-tuning하는 Apache-2.0 통합을 공개했다.
- 최신 발표들은 agent 경쟁력이 model 단독보다 **지속적 post-training, 명시적 상태·증거 관리, 공격 표면 통제, held-out 평가**에 좌우됨을 공통으로 보여준다.

## 읽을거리/추적할 점

- SearchOS의 WideSearch·GISA 성능을 동일 search provider·budget·citation validator 조건에서 재현하고, persistent failure memory가 새로운 query 전략까지 과도하게 억제하는지 확인할 것.
- SEED의 자연어 hindsight skill이 실제 causal improvement인지, 단순 추가 compute·context 효과인지 budget-matched ablation과 held-out tool/domain에서 확인할 것.
- BadWAM류 공격에 대해 future-action consistency detector가 adaptive perturbation과 real-camera noise에서도 견디는지, safety-critical false negative를 측정할 것.
- harness evolution은 탐색에 사용하지 않은 task·repository·model에서 평가하고, 단순 retry·best-of-N과 총 token·wall-clock budget을 맞출 것.
- Hugging Face 권고에 따라 Hub token을 rotation하고 최근 account activity를 검토하며, dataset loader의 remote code·template execution을 sandbox·least privilege·egress control로 제한할 것.
- NeMo Automodel의 Kubernetes 지원 시점, Diffusers 신규 architecture 추가 비용, checkpoint round-trip의 byte/metric 동등성을 추적할 것.
- biosequence watermark는 생성 model·합성 사업자·검사기관 사이 표준화와 false positive/negative, 공격자가 변형한 sequence에 대한 강건성 자료를 기다릴 것.
