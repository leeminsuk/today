# 2026-07-06 AI Daily Briefing

## 핵심 요약

- arXiv 최신 제출분은 주말 영향으로 2026-07-02 항목이 최신이지만, **지속 상태 코딩 에이전트 보안·LLM unlearning 위치 정확도·온라인 안전 모니터링·로컬 실행형 fuzzy function·멀티에이전트 사회적 행동**처럼 운영형 LLM의 신뢰성 이슈가 두드러진다.
- 논문 흐름은 모델 크기 경쟁보다 **하네스, 모니터, 평가 testbed, 에이전트 감사**처럼 실제 배포 후 실패를 줄이는 장치에 집중되어 있다.
- 뉴스에서는 Mechanical Turk 신규 고객 중단, Claude Code 기반 레거시 게임 포팅, Baidu 장문 OCR, Mistral CEO의 프라이버시 발언이 **AI 도구가 노동·코딩·문서 처리·기업 데이터 거버넌스**를 동시에 흔들고 있음을 보여준다.
- NVIDIA는 토큰 규모의 상시 추론 수요를 전제로 AI compute 인프라 투자 파트너십을 강조하며, 생산 AI의 병목이 모델 개발에서 **운영 비용과 AI factory 구축**으로 이동하고 있음을 재확인했다.
